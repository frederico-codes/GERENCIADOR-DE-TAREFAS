# Gerenciador de Tarefas API

API REST desenvolvida com Node.js, TypeScript, Express, Prisma e PostgreSQL para gerenciamento de usuários, times e tarefas.

## Tecnologias

- Node.js
- TypeScript
- Express
- Prisma ORM
- PostgreSQL
- Zod
- JWT
- bcrypt
- Jest
- ESLint
- Prettier
- Docker
